# Dashboard Panels

## Panel 1

### Total Processes Created

Purpose

Displays the total number of Sysmon Process Creation events collected.

---

## Panel 2

### Top 10 Executed Processes

Purpose

Identifies the most frequently executed applications.

---

## Panel 3

### Top Parent Processes

Purpose

Displays parent-child process relationships.

---

## Panel 4

### PowerShell Execution Monitoring

Purpose

Monitors PowerShell execution.

---

## Panel 5

### Command Prompt Execution Monitoring

Purpose

Tracks cmd.exe execution.

---

## Panel 6

### Network Connection Monitoring

Purpose

Displays outbound network connections.

---

## Panel 7

### Registry Modification Monitoring

Purpose

Monitors registry value changes.

---

## Panel 8

### High Integrity Process Monitoring

Purpose

Shows processes running with High/System integrity.

---

## Panel 9

### Suspicious Process Detection

Purpose

Detects commonly abused Windows binaries.

---

## Panel 10

### Process Creation Timeline

Purpose

Visualizes process creation over time.
