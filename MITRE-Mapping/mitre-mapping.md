# MITRE ATT&CK Mapping

## Overview

This project aligns Sysmon detections with the MITRE ATT&CK Framework to demonstrate how endpoint telemetry can be used to identify adversary techniques across the attack lifecycle.

---

# Detection Coverage

| Detection | Sysmon Event | MITRE Technique | ATT&CK ID |
|------------|--------------|----------------|------------|
| Process Creation Monitoring | Event ID 1 | Command and Scripting Interpreter | T1059 |
| PowerShell Execution | Event ID 1 | PowerShell | T1059.001 |
| Command Prompt Execution | Event ID 1 | Windows Command Shell | T1059.003 |
| Suspicious Process Detection | Event ID 1 | System Binary Proxy Execution | T1218 |
| High Integrity Process | Event ID 1 | Abuse Elevation Control Mechanism | T1548 |
| Network Connection Monitoring | Event ID 3 | Application Layer Protocol | T1071 |
| Registry Modification | Event ID 13 | Modify Registry | T1112 |

---

# Detection Philosophy

The detections in this lab focus on identifying attacker behavior rather than relying solely on malware signatures. By monitoring endpoint telemetry generated by Sysmon and correlating events in Splunk Enterprise, the project demonstrates a behavior-based detection approach aligned with the MITRE ATT&CK framework.
