# Executive Security Report

## Project Summary

The Enterprise Detection Engineering & Threat Hunting Lab demonstrates how endpoint telemetry collected using Sysmon can be ingested into Splunk Enterprise to detect, investigate, and monitor Windows security events.

The project simulates a Security Operations Center (SOC) workflow by collecting endpoint logs, developing detection logic using SPL, visualizing events through dashboards, and documenting investigation procedures.

---

# Objectives

- Collect Windows endpoint telemetry
- Build custom SPL detections
- Develop SOC dashboards
- Perform threat hunting
- Document incident response procedures
- Map detections to MITRE ATT&CK

---

# Security Capabilities

- Process Creation Monitoring
- PowerShell Monitoring
- Command Prompt Monitoring
- Network Connection Monitoring
- Registry Monitoring
- High Integrity Process Monitoring
- Suspicious Process Detection
- Threat Hunting
- Incident Response

---

# Technologies

- Splunk Enterprise

# Outcome

The project successfully demonstrates practical Detection Engineering, Security Monitoring, Threat Hunting, and Incident Investigation techniques within a simulated enterprise SOC environment.
