# Lab Setup Process

## Objective

Build a Windows endpoint monitoring environment using Splunk Enterprise and Sysmon.

## Workflow

1. Install Windows 10 VM.
2. Install Sysmon.
3. Install Splunk Enterprise.
4. Configure Splunk Universal Forwarder.
5. Ingest Sysmon logs.
6. Develop SPL queries.
7. Create dashboard panels.
8. Validate detections.
9. Document findings.

## Outcome

A fully functional detection engineering lab capable of monitoring Windows endpoint activity, performing threat hunting, and supporting SOC investigations.
